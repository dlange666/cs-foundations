# 双指针（Two Pointers）

双指针的核心是用两个下标在同一数组或字符串上移动，通过指针间的相对位置减少搜索空间。

## 1. 适用场景

- 有序数组中的查找（左右逼近）
- 去重、原地删除（快慢指针）
- 子串/子数组窗口问题（左右边界扩展与收缩）

## 2. 常见模式

### 2.1 左右指针（对撞指针）

适合有序数组查找目标值。

```python
def two_sum_sorted(nums, target):
    left, right = 0, len(nums) - 1
    while left < right:
        s = nums[left] + nums[right]
        if s == target:
            return [left, right]
        if s < target:
            left += 1
        else:
            right -= 1
    return [-1, -1]
```

时间复杂度 `O(n)`，空间复杂度 `O(1)`。

### 2.2 快慢指针

适合原地过滤、去重。

```python
def remove_element(nums, val):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow  # 新长度，nums[:slow] 为有效区间
```

时间复杂度 `O(n)`，空间复杂度 `O(1)`。

## 3. 解题步骤

1. 明确数组是否有序、是否允许原地修改。
2. 选择指针策略：对撞 / 快慢 / 滑动窗口。
3. 写出指针移动规则，保证每次循环都有推进。
4. 确认边界条件：空数组、单元素、重复元素。

## 4. 易错点

- `while left < right` 与 `while left <= right` 混用导致越界。
- 更新答案后忘记移动指针，导致死循环。
- 快慢指针中误用 `fast` 返回长度。
